Heart Disease Detection

    A machine learning model that can help you determine if you are close to developing a heart disease.

    Heart Disease Prediction

    Please fill in the following form below. You may press Enter to proceed to the next input field
    Chest Pain Type
    • Type 0 if Typical angina: chest pain related decrease blood supply to the heart
    • Type 1 if Atypical angina: chest pain not related to heart
    • Type 2 if Non-anginal pain: typically esophageal spasms (non heart related)
    • Type 3 if Asymptomatic: chest pain not showing signs of disease
    • Source: https://www.kaggle.com/code/faressayah/predicting-heart-disease-using-machine-learning
    Resting electrocardiographic results
    • Type 0 if there's nothing to note
    • Type 1 if it has ST-T Wave abnormality. It can range from mild symptoms to severe problems and signals non-normal heart beat
    • Type 2 for possible or definite left ventricular hypertrophy. Enlarged heart's main pumping chamber
    • Source: https://www.kaggle.com/code/faressayah/predicting-heart-disease-using-machine-learning
    Slope
    • Type 0 if Upsloping: better heart rate with excercise (uncommon)
    • Type 1 if Flatsloping: minimal change (typical healthy heart)
    • Type 2 if Downslopins: signs of unhealthy heart
    • Source: https://www.kaggle.com/code/faressayah/predicting-heart-disease-using-machine-learning
    Thalium stress result
    • Type 1,3 if normal
    • Type 6 if fixed defect: used to be defect but ok now
    • Type 7 if reversable defect: no proper blood movement when excercising
    • Source: https://www.kaggle.com/code/faressayah/predicting-heart-disease-using-machine-learning
